# MovieCatalog
Udemy React Native Cource Movie Catalog

## Using

https://github.com/kristerkari/pinar

## Setup the project
Clone the project
```bash
git clone git@github.com:alinakassym/MovieCatalog.git
```
Install dependencies
```bash
npm install
```

## Run the project
on Android
```bash
npm run android
```

or iOS
```bash
npm run ios
```
